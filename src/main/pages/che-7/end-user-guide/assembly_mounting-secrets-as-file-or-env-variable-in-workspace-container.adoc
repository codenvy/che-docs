= Mounting secrets as file or environment variable into workspace containers (beta)

:context: mounting-secrets-as-file-or-env-variable-in-workspace-container


The following section describes how to prepare secrets to make them automatically mount in workspaces containers as files or environment variables. This can be useful for permanent mount of maven
settings, ssh keys, aws tokens and other. This functionality is using standard secrets mounting
mechanism of Kubernetes, but requires some additional secret annotating and labeling to be properly
bind into {prod} workspace containers.
